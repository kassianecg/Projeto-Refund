# Refund - Solicitação de Reembolso

Aplicação web para registro e controle de despesas para reembolso, desenvolvida com HTML, CSS e JavaScript puro.

## Funcionalidades

- Adicionar despesas informando nome, categoria e valor
- Visualizar a lista de despesas adicionadas com ícone por categoria
- Contagem automática do total de despesas
- Soma total das despesas formatada em Real (R$)
- Remoção individual de despesas com atualização automática dos totais
- Formatação automática do valor em moeda brasileira durante a digitação

## Categorias disponíveis

- Alimentação
- Hospedagem
- Serviços
- Transporte
- Outros

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/kassianecg/Projeto-Refund.git
   ```
2. Abra o arquivo `index.html` no navegador ou utilize a extensão **Live Server** no VS Code.
